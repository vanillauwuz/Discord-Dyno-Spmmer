# Discord-Dyno-Spmmer
 Sending spam messages with the Dyno Bot

# How to use
- Start by setting up the Dyno Bot with the command /tag create name: **the name of the bot you want to use and the command** content: **the message you want Dyno to send**
- In index.ts, enter the token in the **TOKEN** field and the **GUILD_ID** to spam the **DEFAULT_VALUE** command entered in Dyno. Enter the **SPEED** to send in ms
- run index.ts
